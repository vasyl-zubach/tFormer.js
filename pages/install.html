Installation & usage

1. Download source:

Production version (minified) or Developers version

Or clone it from GitHub repo:

git clone https://github.com/TjRus/tFormer.js.git
2. Add sources to your page:
<script src="tFormer.min.js"></script>
3. Define new tFormer and it's rules for your form:
Example
Rules: alpha-numeric, 5 characters length
Rules: required, email
Validation timeout - 500
Controlled by tFormer 
<form id="my_form_id">
	<input type="text" name="zip" />
	<input type="text" name="email" />
	<input type="submit" value="Submit" />
</form>
<script>
var my_form_id = new tFormer('my_form_id',{
	fields: {
		zip: 'a1 l=5', // define only rules
		email: { //defining rules and other options
			rules: '* @',
			timeout: 500
		}
	}
});
</script>

Defining validation rules

data-rules

Define all rules in data attribute - data-rules:

<form id="my_form_id">
	<input type="text" name="zip" data-rules="a1 l=5" />
	<input type="text" name="email" data-rules="* @" />
	<input type="submit" value="Submit" />
</form>
<script>
var my_form_id = new tFormer('my_form_id');
// or if we need to define custom options
var my_form_id = new tFormer('my_form_id', {
	errorClass: 'my_own_errorClass',
	// other options
	fields: {
		email: {
			timeout: 500 // timeout form email field validation
		}
	}
});
</script>
tFormer options

Define all rules in whide defining new tFormer object for the form:

<form id="my_form_id">
	<input type="text" name="zip" />
	<input type="text" name="email" />
	<input type="submit" value="Submit" />
</form>
<script>
var my_form_id = new tFormer('my_form_id', {
	fields: {
		zip: 'a1 l=5', // simple string, if there is no need to define other options fot this field
		email: { // defining rules in object with other field options
			rules: '* @',
			timeout: 500 // timeout form email field validation
		}
	}
});
</script>
